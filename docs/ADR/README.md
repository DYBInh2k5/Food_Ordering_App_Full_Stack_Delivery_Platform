# ADR Index

This folder contains Architecture Decision Records (ADR).

## How to Add a New ADR
1. Copy 000-template.md.
2. Rename to next number, for example 001-api-versioning.md.
3. Fill all sections with concrete context and decision details.
4. Open a pull request and request technical review.

## Naming Convention
- Use zero-padded numeric prefix.
- Use short kebab-case title.
- Example: 002-payment-webhook-idempotency.md

## Status Values
- proposed
- accepted
- superseded
- deprecated
